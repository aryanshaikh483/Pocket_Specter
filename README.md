# Pocket Specter — AI Legal Assistant for Indian Law

> **Abstract** : Pocket Specter is an AI-powered legal assistant that simplifies access to Indian law through multilingual, context-aware interaction. It enables users to generate FIRs, RTI applications, and legal documents while providing real-time legal guidance across multiple domains. Built using a full-stack architecture , the system also supports document-based Q&A and persistent memory. By reducing language and cost barriers, Pocket Specter aims to make legal assistance fast, affordable, and accessible for all.

### Deployment Steps

Please follow the below steps to run this project.

1. Clone the repository
2. Install dependencies (`npm install`)
3. Configure environment variables (MongoDB URI, Groq API key, Clerk keys)
4. Run backend server (`npm run server`)
5. Run frontend (`npm run dev`)
6. Open [http://localhost:3000](http://localhost:3000)
